# 산모수첩 (Maternity Notebook)

산모와 아기를 위한 기록 및 관리 앱 프로젝트입니다. Compose Multiplatform을 사용하여 Android 및 iOS 앱으로 개발됩니다.

## 프로젝트 구조

- `cmp/`: Compose Multiplatform 기반의 앱 소스 코드가 포함된 메인 개발 폴더입니다.
- `design/`: 앱의 디자인 문서, 리소스(아이콘, 이미지 등) 및 UI/UX 기획서가 포함된 폴더입니다.
- `docs/`: 프로젝트 관련 추가 문서들이 위치하는 곳입니다.

## 주요 기능 (예정)

- 산모 건강 기록 (체중, 혈압 등)
- 태아 성장 기록 및 초음파 사진 저장
- 진료 일정 및 알림
- 출산 준비 체크리스트
- 태아에게 남기는 일일 편지
